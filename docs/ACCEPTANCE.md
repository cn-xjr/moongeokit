# 验收证据

## 正确性

- 基础几何、凸包、点包含和折线算法均有确定性测试。
- 网格索引覆盖点查询、范围查询、跨桶去重和越界输入。
- 容差谓词覆盖严格与宽松 epsilon。
- JavaScript、Wasm、Wasm-GC 本地测试一致；CI 额外验证 Native。

## 性能模型

| 操作 | 时间复杂度 |
| --- | --- |
| 线性索引查询 | O(n) |
| 网格点查询 | O(k) |
| 网格范围查询 | O(b + k) |
| 插入 | O(c) |

`b` 是查询覆盖的网格数，`k` 是去重后的候选数，`c` 是对象覆盖的网格数。

运行 `moon run cmd/bench` 会构建 10,000 个规则分布对象。线性基线每次查询需要检查
10,000 个对象，网格索引直接输出实际 `candidates_scanned` 和 `buckets_visited`。
