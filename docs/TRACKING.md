# 公开开发跟踪

MoonGeoKit 按公开仓库持续开发方式推进，核心证据包括提交记录、Issue、Pull Request、CHANGELOG、ROADMAP 和 CI。

## 当前提交主题

- 几何核心类型
- 线段相交检测
- AABB 包围盒
- 多边形面积与周长
- 点在多边形内
- 点集凸包
- 折线长度与简化
- 点到线段距离
- JSON 导出
- CLI 演示
- CI 与协作模板
- README、路线图和更新日志

## 后续工单建议

1. 支持 RDP 折线简化
2. 支持多边形凸性判断
3. 支持 GeoJSON 导入导出
4. 增加最近点对算法
5. 增加 WebAssembly 示例
